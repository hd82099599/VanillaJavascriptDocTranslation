# Contribute
 - 若您在已翻译项目中发现错误或不理解的地方，请提交Issue
 - 为本项目贡献，请用Github标准工作流程。（PR）
```
 1.英语请过关
 2.不要过度使用翻译工具
 3.不要碰别人的翻译，如有必要请提交Issue
 4.可以在翻译条目前添加自己的名字
 5.不合规的PR会被Close 望悉知。
```
